## 1. Генерация текста медитаций (LLM)

### Основные провайдеры:

#### OpenAI GPT-4/GPT-4 Turbo
- **API**: https://platform.openai.com/docs/api-reference
- **Особенности**: Отличное понимание контекста, поддержка русского языка
- **Цена**:

- **Особенности**: Специально настроен для медитаций, правильная обработка пауз, счет, эмоциональные переходы. Обработка многоточий как пауз (0.333 сек)

- **Простота интеграции**: ⭐⭐⭐ (3/5) - Может потребовать дополнительной информации
- **Рекомендация**: ✅ Лучший выбор для медитаций, если доступен

#### ElevenLabs
- **API**: https://elevenlabs.io/docs/api-reference
- **Особенности**: Эмоциональные голоса, клонирование голосов
- **Цена**: 
  - Starter: $5/месяц (30K символов)
  - Creator: $22/месяц (100K символов)
  - Pro: $99/месяц (500K символов)
- **Качество**: ⭐⭐⭐⭐⭐ (5/5) - Высокое качество, естественные голоса
- **Простота интеграции**: ⭐⭐⭐⭐⭐ (5/5) - Отличное API
- **Рекомендация**: ✅ Топ выбор для озвучки

#### Hume AI
- **Особенности**: Эмоциональный интеллект в голосе, передача эмоций и настроения
- **Качество**: ⭐⭐⭐⭐ (4/5) - Уникальная эмоциональная составляющая
- **Простота интеграции**: ⭐⭐⭐⭐ (4/5)
- **Рекомендация**: ✅ Рассмотреть для эмоциональной озвучки

#### OpenAI TTS
- **API**: https://platform.openai.com/docs/guides/text-to-speech
- **Модели**: tts-1, tts-1-hd
- **Голоса**: alloy, echo, fable, onyx, nova, shimmer
- **Цена**:

- **Качество**: ⭐⭐⭐⭐ (4/5) - Стабильное качество
- **Простота интеграции**: ⭐⭐⭐⭐⭐ (5/5) - Простая интеграция с OpenAI

## Стоимость и лимиты

### Бесплатные тарифы:

- **OpenAI**: $5 кредитов при регистрации
- **Google Cloud**: $300 кредитов на 90 дней
- **Mubert**: Ограниченный бесплатный план
- **ElevenLabs**: 10K символов бесплатно/месяц
- **Recombee**: До 100K событий/месяц бесплатно
- **Firebase**: Spark план с ограничениями
- **Supabase**: 500MB базы данных бесплатно

### Оценка месячных затрат (MVP, ~1000 активных пользователей):

- OpenAI GPT-4 Turbo: ~$50-100 (зависит от использования)
- ElevenLabs: $22-99 (Creator или Pro план)
- Mubert: $14-50 (в зависимости от плана)
- Firebase/Supabase: $0-25 (зависит от использования)
- **Итого**: ~$86-274/месяц

### Рекомендации по оптимизации:

1. **Кэширование**: Сохранять популярные медитации, не генерировать каждый раз
2. **Локальные модели**: Использовать TensorFlow Lite для простых задач
3. **Батчинг**: Группировать запросы к API где возможно
4. **Мониторинг**: Отслеживать использование квот и оптимизировать запросы
5. **A/B тестирование**: Тестировать разные модели для оптимизации затрат
6. **CDN**: Использовать CDN для кэширования аудио файлов
